# v0.33 Official MCP Preview CN

`v0.33.0-preview` 不是普通 `main` MCP 结构。它是用于验证 UE 5.8 first-party `ToolsetRegistry + ModelContextProtocol` 的 experimental branch。

## 结构

| Track | Endpoint | Role |
| --- | --- | --- |
| UEAtelier track | `127.0.0.1:8765/mcp` | 现有 `unreal.*` tool surface、policy、audit、Task Atlas、RAG、self-extension。 |
| UE 5.8 official track | `127.0.0.1:8000/mcp` when enabled | official `ToolsetRegistry + ModelContextProtocol`。stateful streamable HTTP。 |
