# v0.33 Official MCP Preview JP

`v0.33.0-preview` は通常の `main` MCP 構造ではありません。UE 5.8 の first-party `ToolsetRegistry + ModelContextProtocol` を検証する experimental branch です。

## 構造

| Track | Endpoint | Role |
| --- | --- | --- |
| UEAtelier track | `127.0.0.1:8765/mcp` | 既存の `unreal.*` tool surface、policy、audit、Task Atlas、RAG、self-extension。 |
